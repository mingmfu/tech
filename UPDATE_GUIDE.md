## 🔄 更新要求

### 1. AI热点新闻（20条）
- **国内新闻**：10条，基于真实国内AI动态
- **国际新闻**：10条，基于真实国际AI动态  
- **日期**：2026年2月最新
- **摘要**：260+字，完整无省略
- **链接**：指向真实新闻来源

### 2. 前沿技术（OS、PL、GFX）
- **数量**：每栏目6条
- **日期**：2026年2月最新技术发布
- **摘要**：260+字
- **链接**：指向官方发布页面或技术博客

## 🚀 更新方式

### 方式1：手动更新
```bash
cd ~/tech
python3 update_all.py
```

### 方式2：自动更新（GitHub Actions）
- 每天北京时间10:00自动更新
- 触发地址：https://github.com/mingmfu/tech/actions/workflows/daily-update.yml
- 点击"Run workflow"可手动触发
