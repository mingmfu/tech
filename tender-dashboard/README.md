### Step 3: 遍历采集

对每个项目执行：

#### 3.1 进入项目详情

1. 点击项目名称或"查看详情"
2. 等待页面加载完成
3. 确认项目地点包含"安徽"
4. 确认项目标题包含"审计"

#### 3.2 提取项目信息

**基本信息**（4个）
- `项目名称`：完整标题
- `招标编号`：如有则提取
- `发布日期`：YYYY-MM-DD格式
- `截止时间`：YYYY-MM-DD HH:MM格式

**单位信息**（4个）
- `招标单位`：招标人全称
- `代理机构`：招标代理全称
- `联系人`：项目负责人姓名
- `联系方式`：电话、邮箱（可部分隐藏）

**项目详情**（6个）
- `项目金额`：预算金额（单位：万元）
- `项目地点`：项目实施地点
- `项目概况`：项目背景、目标（100字内）
- `服务范围`：审计服务具体内容
- `服务期限`：服务周期
- `质量标准`：服务质量要求

**投标要求**（6个）
- `资质要求`：投标人资质证书
- `业绩要求`：类似项目经验
- `人员要求`：团队人员配置
- `投标保证金`：金额及缴纳方式
- `文件售价`：招标文件价格
- `递交方式`：投标文件递交方式

**评标信息**（3个）
- `评标方法`：综合评分法/最低评标价法等
- `定标方法`：如何确定中标人
- `评审专家人数`：如有则记录

**其他信息**（3个）
- `公告原文链接`：当前页面URL
- `附件列表`：招标文件等附件名称
- `备注`：其他重要信息

#### 3.3 返回列表继续

1. 点击浏览器"返回"按钮
2. 确保回到项目列表页
3. 重复Step 3，处理下一个项目

**注意事项**：
- 单页超时30秒则跳过该项目
- 项目已截止则跳过
- 每处理5个项目，报告进度

### Step 5: 智能推荐

基于6个维度进行评分（满分100分）：

#### 评分维度

**1. 金额匹配度（20分）**
- 18-20分：10-50万元（理想区间）
- 12-17分：50-100万元
- 5-11分：<10万元或>100万元
- 0-4分：金额未明确

**2. 时间窗口（20分）**
- 18-20分：10-20天（理想）
- 12-17分：5-10天
- 5-11分：20-30天或<5天
- 0-4分：>30天

**3. 资质匹配（20分）**
- 18-20分：完全匹配，所有要求可满足
- 12-17分：基本匹配，个别次要要求不满足
- 5-11分：部分匹配，存在短板
- 0-4分：不匹配，关键资质缺失

**4. 历史中标率（15分）**
- 13-15分：该类型/业主历史中标率高（>50%）
- 8-12分：历史中标率中等（20-50%）
- 3-7分：历史中标率低（<20%）
- 0-2分：无历史数据

**5. 竞争程度（15分）**
- 13-15分：竞争程度低，预估投标人数<5家
- 8-12分：竞争程度中等，预估5-10家
- 3-7分：竞争激烈，预估>10家
- 0-2分：竞争异常激烈

**6. 客户价值（10分）**
- 8-10分：央企、国企总部、知名上市公司
- 5-7分：地方国企、一般上市公司
- 2-4分：民营企业、事业单位
- 0-1分：其他

## Constraints

### 执行约束

1. **验证码处理**：
   - 如遇图形验证码，停止并报告
   - 报告："遇到验证码，无法自动登录，需要人工处理"

2. **超时处理**：
   - 单页加载超时：30秒未完成加载则跳过
   - 报告："项目[名称]加载超时，已跳过"

3. **日期检查**：
   - 只采集**未截止**的项目
   - 已截止项目跳过

4. **地区检查**：
   - 确保项目地点在安徽省内

5. **数据完整性**：
   - 关键字段缺失过多则标记为"部分提取"

6. **隐私保护**：
   - 联系人电话/邮箱可部分隐藏（如：138****1234）

### 输出约束

1. **JSON格式**：
   - 必须输出有效的JSON格式
   - 中文字段名使用中文

2. **文件保存**：
   - 保存到 `tender_collection_[日期时间].json`
